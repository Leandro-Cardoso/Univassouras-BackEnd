🔧 Pré-requisitos

  • Python: Download Python
    • Abra o terminal (no Windows: pesquise por "Prompt de Comando" ou "PowerShell") e digite: python --version para conferir a instalação.
  • Git: Download Git
    • Verifique no terminal com: git --version
  • Visual Studio Code: Download VSCode

🌐 Git e GitHub

Git é o caderno de anotações do seu código, GitHub é a nuvem onde você pode guardar e compartilhar esses cadernos.
  • Crie uma conta no GitHub: https://github.com/
  • Criar repositório:
    • Depois de logar no GitHub → clique em New → dê um nome → clique em Create repository
    • Copie o link HTTPS, que será usado para baixar o repositório no seu computador.

📁 Clonar Repositório

"Clonar" é como copiar todo o caderno de alguém para o seu computador.
  • Crie uma pasta no computador para o projeto (ex.: Área de Trabalho → Pasta "MeusProjetos").
  • No VSCode: View → Command Palette → Clone Git Repository → cole o link HTTPS → escolha a pasta criada.

📂 Criar pasta do Projeto

  • Dentro da pasta clonada, crie uma subpasta com um nome fácil de lembrar, ex.: meu_primeiro_projeto

🛠️ Criar Ambiente Virtual

O ambiente virtual é como uma caixa de ferramentas só para este projeto, sem misturar com outros projetos.
  • Abra o terminal dentro da pasta do projeto.
  • Digite: python -m venv .venv
  • Isso cria a pasta .venv que vai guardar todas as bibliotecas que o projeto precisa.

🔄 Ativar o Ambiente Virtual

  • No terminal (CMD): .\.venv\Scripts\activate
  • No PowerShell, se der erro: Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
  • Depois de ativar, o terminal vai mostrar algo como (.venv) antes do prompt.

📥 Instalar Django

Django é o "motor" que vai ajudar a criar aplicações web rapidamente.
pip install django

🚀 Criar Projeto Django

  • No terminal, ainda dentro da pasta do projeto, digite: django-admin startproject setup .
  • Isso cria a estrutura inicial do Django com pastas e arquivos básicos.

🏃 Executar o Projeto

  • No terminal: python manage.py runserver
  • Acesse no navegador: http://localhost:8000
  • Para ver o site em português: abra setup/settings.py e altere LANGUAGE_CODE = 'pt-br'

🛠️ Resolução de Problemas

  • Se não conseguir ativar o ambiente virtual: verifique se digitou o comando corretamente e ajuste a política do PowerShell.
  • Se Django não instalar: confira se o ambiente virtual está ativado antes de usar pip install django.
